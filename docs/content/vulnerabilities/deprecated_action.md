# Deprecated Action

## Vulnerability Description


Action {uses} uses v1 version, which is often deprecated.
Check if a newer version is available with security fixes and improvements.


## Recommendation


Check if a newer version of this action is available:


1. Visit the action repository:

https://github.com/{action_name}


2. Check for newer versions:

- Review releases page

- Check for security advisories

- Look for v2, v3, or later versions


3. Update to the latest version:

- Use the latest stable version

- Pin to commit SHA for maximum security

- Review changelog for breaking changes
