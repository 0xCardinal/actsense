# Typosquatting Action

## Vulnerability Description


Action {uses} has suspicious pattern: {description}. This might indicate typosquatting:

- Typosquatting attacks use similar names to trick users

- Malicious actions can compromise your workflows

- Attackers rely on users not noticing the difference


Security risks:

- Potential typosquatting attack

- Malicious code execution

- Secret exfiltration

- Supply chain compromise


## Recommendation


Verify the action is legitimate:


1. Review the action repository:

- Visit: https://github.com/{action_name}

- Check repository activity and maintenance

- Review code and security posture


2. Verify the publisher:

- Confirm the publisher is trusted

- Check for security advisories

- Review repository history


3. If suspicious:

- Use the correct/official action instead

- Report suspicious actions to GitHub

- Review all actions in your workflows
